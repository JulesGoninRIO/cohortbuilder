src.discovery.definitions
=========================

.. automodule:: src.cohortbuilder.discovery.definitions

In the status classes, the value of the enumerations indicates the sequence of the statuses.
The whole numbers show the expected itenary of the object status.
The fractional numbers show the alternatives at the same level.
