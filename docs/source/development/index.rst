Development
================

In order to develop Cohort Builder, it is suggested to work
on the development branch.
You can pull the development branch with the following commands:

.. code:: bash

    git switch development
    git pull

Before starting to contribute to the project, please read all the following
pages and try to keep them updated in the course of the development:
